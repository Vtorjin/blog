# start redis

  • 必须在命令行里面启动redis,否则会报错,启动code:cd D:\Program Files\Redis
  • 使用桌面程序启动 cd D:\Program Files\Redis && redis-server.exe redis.windows.conf
  • 桌面启动后,该redis服务无法进行直接操作,需要使用redis-cli进行操作管理,启动code cd D:\Program Files\Redis && redis-cli

# handle redis

可以在 redis.windows.conf文件中设置maxheap（最大内存）,maxmemory(最大内存),**requirepass(通信密码)**等内容

  • 密码验证: cli下使用 auth 'your redis passwrod';
  • 设置: set props value,得到在操作结果是 OK
