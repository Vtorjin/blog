# 数据结构

# 栈结构

# 是什么?

栈(Stack)是一种线性存储结构，它具有如下特点： 栈中的数据元素遵守”后进先出”(First In Last Out)的原则，简称FILO结构。 限定只能在栈顶进行插入和删除操作。

class Stack {
    constructor() {
        this.items = []
    }
    push(item) {
        this.items.push(item)
    }
    pop() {
        return this.items.pop()
    }
    // 栈顶元素
    peek() {
        return this.items[this.items.length - 1];
    }
    // 栈顶是否为空
    isEmpty() {
        return this.items.length === 0
    }
    size() {
        return this.items.length
    }
    toString() {
        let string = "";
        return this.items.reduce((res, item) => res + item, string)
    }
}
  • push（element）：添加一个新元素到栈顶位置；
  • pop（）：移除栈顶的元素，同时返回被移除的元素；
  • peek（）：返回栈顶的元素，不对栈做任何修改（该方法不会移除栈顶的元素，仅仅返回它）；
  • isEmpty（）：如果栈里没有任何元素就返回true，否则返回false；
  • size（）：返回栈里的元素个数。这个方法和数组的length属性类似；
  • toString（）：将栈结构的内容以字符串的形式返回。

# 栈结构应用场景：

  • 文本编辑器、绘制程序中的撤销功能
  • 函数调用堆栈
  • Web 浏览器所访问过的链接的历史记录
  • 判断字符串的括号是否有效

# 环形队列

# 是什么?

循环队列是一种线性数据结构，其操作表现基于 FIFO（先进先出）原则并且队尾被连接在队首之后以形成一个循环。它也被称为“环形缓冲器”。

循环队列的一个好处是我们可以利用这个队列之前用过的空间。在一个普通队列里，一旦一个队列满了，我们就不能插入下一个元素，即使在队列前面仍有空间。但是使用循环队列，我们能使用这些空间去存储新的值。

# 环形队列应用场景

  • Memory Management: The unused memory locations in the case of ordinary queues can be utilized in circular queues.
  • Traffic system: In computer controlled traffic system, circular queues are used to switch on the traffic lights one by one repeatedly as per the time set.
  • CPU Scheduling: Operating systems often maintain a queue of processes that are ready to execute or that are waiting for a particular event to occur. 操作系统通常维护一个进程队列，这些进程已经准备好执行，或者正在等待特定事件的发生。
  • Lock Free Queue: When high performance is required, we don't want to use lock. Circular Queue can is the top 1 data structure for lock free queue.

    constructor(capacity) {
        this.queue = new Array(capacity); //生成队列长度
        this.queueCapacity = capacity;
        this.queueHead = 0; //队头
        this.queueTail = 0; //队尾
        this.queueLength = 0; //队列长度
    }

    clearQueue() {
        this.queueHead = 0;
        this.queueTail = 0;
        this.queueLength = 0;
    }

    isQueueEmpty() {
        return this.queueLength === 0
    }

    isQueueFull() {
        return this.queueCapacity === this.queueLength
    }

    getQueueLength() {
        return this.queueLength;
    }

    enQueue(el) {
        if (this.isQueueFull()) {
            console.warn('入队失败;队列已满；入队元素:' + el);
            return false;
        }
        //队尾插入
        this.queue[this.queueTail] = el;
        this.queueTail++;
        this.queueTail = this.queueTail % this.queueCapacity; // 需要对队列容量做取余操作，才能实现环形队列
        this.queueLength++;
        console.log('入队成功 入队元素为:' + el);
        return true;
    }

    deQueue() {
        if (this.isQueueEmpty()) {
            console.warn('出队失败，对列为空');
            return false;
        }
        //获取队头
        const el = this.queue[this.queueHead];
        this.queueHead++;
        this.queueHead = this.queueHead % this.queueCapacity; //对队列容量做区域操作，才能实现环形队列
        this.queueLength--;
        console.log('出队成功；出队元素为' + el);
        return el;
    }

    traverseQueue() {
        var queuesStr = '',
            i =  this.queueHead,
            len = this.queueHead + this.queueLength;
        // 从队头开始，次数为队头下标加上队长度, 保证循环次数不受队头位置变化影响

            queuesStr += element + ' ';
        }

    }
}
