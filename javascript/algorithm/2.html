# 算法系列之: 删除有序数组中的重复项

要求: 给你一个升序排列 的数组 nums ，请你原地(原有的地址)删除重复出现的元素，使每个元素只出现一次,返回删除后数组的新长度。元素的相对顺序应该保持一致. 示例:

input: [1,2,2,5,5,6,6];
output: 4, //原数组[1,2,5,6]; 不需要考虑数组中超出新长度后面的元素。

个人思路: 考虑到是在原有的地址上进行删除,因此常用的Set去重方案就不能在使用了. 因此想到的就是在循环时当前的index后和后面的index(+1)下进行比对,有重复的就splice一下. 所以当时的代码就是:

var  removeDuplicates = function(nums) {
    if(nums.length<2) return nums.length; 
    var repeat = 0,i=0;
    while(true){
        if(i >= nums.length) break;
        nums[i] === nums[i+1] ? (nums.splice(i,1), i--) : repeat++;
        i++;
    }
    return repeat
};

总结: 这段代码的执行效率并不是高(1100ms),循环当前index和index+1一致时splice的方案不是最优的

优秀方案:

方案1: 声明一个空的object,在--循环的时候对数组中第一次出现的值进行记录,已经出现的值进行splice处理. 整体的测试时间在60-70ms之间

var removeDuplicates = function (nums=[]) {
   if(nums.length < 2) return nums.length;
   let obj ={},i = nums.length;
   while(true){
        if(i < 0 ) break;
        let  key = nums[i];
        if(!obj[key]){
            obj[key]=1;
        }else{
            nums.splice(i,1);
        }
        i--;
    }
    return nums.length;
};

方案2: 由于是排序后的数据,可以使用数组中的lastIndexOf(数组中同一个数值出现的index位置),获取到最后一个数值的lastIndex后，数组只需要保留一位数值即可,测试时间在 70-80m之间

var removeDuplicates = function (nums=[]){
   if(nums.length < 2) return nums.length;
    let i = 0;
    while(true){
        if(i >=nums.length) break;
        const lastIndex = nums.lastIndexOf(nums[i]);
        //这里妙在的是lastIndex的值可以是-1，而splice的的第二个值是负数时，对原数组是不产生影响的
        nums.splice(i, last Index-i);
        i++;
    }
    return nums.length;
}
