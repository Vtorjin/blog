# Web Workers

web workers是什么?

h5规范提出了web workers,可以让js在分线程中执行

如何使用?

// main thread:index.js 
var woker = new Worker('worker.js'); //your handle another thread js-file path;

// received message from another thread;
worker.onMessage = function(event){
    const message = event.data; //received message;
}
//another thread:worker.js
worker.postMessage('i told you some new messages!');

缺陷

  • worker内不能操作dom更新UI
  • 不是每个浏览器都支持这个新特性
  • 不能跨域加载JS
