    # Javascript中JSON隐藏的特性

    在日常的开发中,JSON对象用的最多的也就是**JSON.stringify()将对象或数组转为字符串和JSON.parse()**将JSON格式的字符串转为对象两个方法了,但是除此之外,JSON对象也还有其他的一些特性.

    # 在转换为字符串的时候隐藏某个属性

    JSON.stringify叫做 replacer, 它是个用于决定哪些数据输出哪些数据不输出的函数或者数组

    以下示例可以隐藏用户的 class 字段：

    const user = {
  name: 'John',
  password: '12345',
  age: 30,
  class:['senior developer','junior developer','intermediate developer']
};

console.log(JSON.stringify(user, (key, value) => {
    if (key === 'class') {
            return;
    }

    return value;
}));

// 优化版
const stripe =  (...keys) => {
  return (key, value) =>{
      if(keys.includes(key)){
        return
      }
      return value;
  }
}

JSON.stringify(user, stripe('name', 'class')); //'{"password":"12345","age":30}'
JSON.stringify(user, stripe(['name', 'class']))

    stringify的第二个参数也可以是数组,也能实现同样的效果

    // 隐藏recipe属性
const cakes = [
    {
        name: 'Chocolate Cake',
        recipe: [
            'Mix flour, sugar, cocoa powder, baking powder, eggs, vanilla, and butter',
            'Mix in milk',
            'Bake at 350 degrees for 1 hour',
            // ...
        ],
        ingredients: ['flour', 'sugar', 'cocoa powder', 'baking powder', 'eggs', 'vanilla', 'butter']
    }

];
console.log(JSON.stringify(cakes,['ingredients','name']))
//[{"ingredients":["flour","sugar","cocoa powder","baking powder","eggs","vanilla","butter"],"name":"Chocolate Cake"}]

    # 使用 toJSON 创建自定义输出格式

    如果对象已经实现了 toJSON 方法, JSON.stringify 将使用该方法格式化数据

    // 没有toJSON()方法
class Fraction {
  constructor(n, d){
    this.numerator = n;
    this.denominator = d;
  }
}
JSON.stringify(new Fraction(1,2)); //'{"numerator":1,"denominator":2}'

//定义了toJSON()方法
class Fraction {
  constructor(n, d){
    this.numerator = n;
    this.denominator = d;
  }
  toJSON(){
    return `${this.numerator}/${this.denominator}`
  }
}

//JSON.stringify 将遵循 toJSON 属性并输出字符串 "1/2".
JSON.stringify(new Fraction(1,2)); //'"1/2"'

    # 恢复自定义输出格式后的数据

    关键点就是判断当前的值是否是自定义的格式的数据,并传递第二个参数给 JSON.parse 去指定恢复函数。 恢复函数的工作是恢复已转换为字符串的数据为原始格式

    class Fraction {
  constructor(n,d){
    this.numerator = n;
    this.denominator = d;
  }
  // toJSON()自定义格式数据
  toJSON(){
    return `${this.numerator}/${this.denominator}`
  }

  static recover(key,value){
    // 仅需对value进行操作,当前的key为空
    if(typeof value === 'string'){
      console.log(key,'-',value);
       //map(Number)-字符串数据转换为数字, map(parseInt)?
        const parts = value.split('/').map(Number);
        //使用...运算符,避免denominator为undefined
        if(parts.length === 2) return new Fraction(...parts);
    }
    return value;  
  }
}

const fraction = new Fraction(1,2);
const stringified = JSON.stringify(fraction); // '1/2'

// Fraction {numerator: 1, denominator: 2}
const revived = JSON.parse(stringified, Fraction.recover); 
console.log(revived)

    附: map(parseInt) = map((item,index) => parseInt(item,index)); //进制不同的数据返回处理的数据都是NaN.

    # 通过转换器隐藏数据

    和替换器(stringify)类似，转换器(parse)也可以用于隐藏数据，方法是一样的.

    const user = JSON.stringify({
  name: 'John',
  password: '12345',
  age: 30
});

JSON.parse(user, (key, value) => {
    if (key === 'password') {
            return;
    }
    return value;
}); // { name: 'John', age: 30 }
