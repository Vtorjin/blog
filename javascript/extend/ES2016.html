# ECMAScript历代版本特性 - ES7(2016)

# Array.prototype.includes()

作用: 用来判断一个数组是否包含一个指定的值，根据情况，如果包含则返回 true，否则返回false。

语法: arr.includes(valueToFind[, fromIndex]);

注意事项:

  • 如果 fromIndex 大于等于数组的长度，则会返回 false
  • 如果 fromIndex 为负值，计算出的索引将作为开始搜索 valueToFind 的位置。如果计算出的索引小于0，则整个数组都会被搜索。
  • includes() 方法有意设计为通用方法。它不要求 this 值是数组对象，所以它可以被用于其他类型的对象（比如类数组对象）。
// array length is 3
// fromIndex is -100
// computed index is 3 + (-100) = -97
const arr = ['a', 'b', 'c'];
arr.includes('c', 3);   // false
arr.includes('c', 100); // false
arr.includes('a', -100); // true
arr.includes('b', -100); // true
arr.includes('c', -100); // true
arr.includes('a', -2);   // false
(function() {
  console.log([].includes.call(arguments, 'a')); // true
  console.log([].includes.call(arguments, 'd')); // false
})('a', 'b', 'c');

# 求幂运算符(**)

** 是一个用于求幂的中缀运算符,和Math.pow(x, y)有同样的效果.

console.log(2 * 3 ** 2); // 2*3*3 = 18
console.log(2 ** 3 ** 2); // 2**(3*3) = 8*8*8 = 512
console.log(2 * 3 ** 3); // 2的27次方 = 134217728
